Die OsmAnd-App ist mit einer großen Anzahl von Standardfunktionen ausgestattet, es besteht jedoch die Möglichkeit, noch mehr hinzuzufügen. Es stehen häufig kostenlose Erweiterungen zur Verfügung, die in bestimmten Situationen helfen sollen. Sie können ein Tool für bessere Parkmöglichkeiten, Wintersportorte, Karten usw. herunterladen. Hier werden einige davon kurz besprochen.

Beginnen Sie mit OsmAnd

OsmAnd bietet Ihnen großartige Möglichkeiten zur Verwendung der Karten. Um mit OsmAnd zu beginnen, können Sie die Anwendung von Google Play oder AppStore herunterladen.

WEITERLESEN
Installation und Fehlerbehebung
Bei bestimmten Problemen können Sie versuchen, die Anwendung erneut zu installieren. Es ist leicht. Zunächst müssen Sie Ihre Dateien sichern. Finden Sie dazu Ihren OsmAnd-Ordner auf Ihrem Gerät und kopieren Sie die Dateien. Oder Sie können sie exportieren.
WEITERLESEN
Kartenansicht
OsmAnd bietet Ihnen großartige Möglichkeiten zur Verwendung der Karten. Sie können sie anpassen, verschiedene Kartentypen herunterladen und sogar bearbeiten!
WEITERLESEN
Suche auf der Karte
Möglicherweise möchten Sie den nächstgelegenen Standort eines bestimmten Typs suchen, Ihren eigenen Standort anzeigen oder einen Ort anhand der Postleitzahl suchen. Nutzen Sie einfach die Anleitung unten.
WEITERLESEN
Reise planen
Sie können eine neue Route mit der Navigationsoption der App erstellen. Dieses Tool soll Ihnen helfen, eine optimale Route gemäß Ihrem Fahrzeug, Ihren persönlichen Bedürfnissen usw. zu finden. Sie können einen Auto-, Fahrrad- oder Fußgängermodus wählen, bestimmte Straßentypen meiden usw. Sobald die Route erstellt ist, Sie erhalten eine Turn-by-Turn-Navigation und Sprachführung.
WEITERLESEN

Navigation

Mit der Navigationsfunktion können Sie Ihr Ziel bequem mit Hilfe von der Sprachführung erreichen (fakultativ).

WEITERLESEN

ERWEITERUNGEN

Online-Karten

Mit dem Online-Karten-Erweiterung können Sie neue Informationen Schichten in Ihre Karte einfügen. Wählen Sie die Kartenquelle selbst aus und fügen Sie der Hauptansicht der Karte bis zu zwei Ebenen hinzu. Das Plugin ist für kleinere Gebiete hilfreich, wenn Sie keine vollständige Landkarte des Landes benötigen.

Höhenlinien und Relief-Schattierungen

Mit dieser Erweiterung können Sie Informationen zu den Höhen des Geländes (Höhenlinien) anzeigen und deren Visualisierung durch Aktivieren der Relief-Schattierungen anzeigen. Wenn Sie diese beiden Funktionen kombinieren, erhalten Sie sowohl die Informationen in Zahlen als auch die visuelle Darstellung. Regelmäßige, Kontur- und Relief-Schattierungen-Karten sind drei verschiedene Arten von Downloads.

Streckenaufzeichnung

Mit dieser Funktion können Sie Ihre Reise aufzeichnen und als GPX-Datei speichern. Es verwendet das GPS Ihres Handys, um Ihre Bewegung zu verfolgen. Sie können die zurückgelegte Route anzeigen, an das OpenStreetMap-Projekt senden oder in externe Dienste hochladen.

WEITERLESEN

Seekarten

Genau wie übliche OpenStreetMap-Karten sind Seekarten Open-Source-Karten. Sie geben Ihnen Informationen über Meere, Flüsse, Häfen usw., einschließlich der Segelrouten, gefährlicher Gebiete, Bereiche, in denen das Segeln oder Docken erlaubt ist oder nicht erlaubt ist usw. Sie sind kein Ersatz für offizielle Seekarten, die Segler auf Schiffen haben müssen. Sie können aber ein großartiges zusätzliches Werkzeug sein.

WEITERLESEN

OSM-Bearbeitung

Möchten Sie die Karte selbst bearbeiten? Werfen Sie einen Blick auf die OSM-Bearbeitungs-Erweiterung. OsmAnd verwendet die Karten, die im Projekt OpenStreetMap erstellt wurden, eine kollaborative Gemeinschaft von Freiwilligen, die Karten erstellen und aktualisieren. Wenn Sie es selbst ausprobieren möchten, aktivieren Sie einfach die Erweiterung und aktualisieren Sie die Informationen über die Points of Interest an einem beliebigen Ort. Um weitere Möglichkeiten zu erhalten, nehmen Sie an OpenStreetMap teil.
