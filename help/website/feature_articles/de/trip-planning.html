Mit OsmAnd können Sie Ihre Reise planen, neue Routen bauen oder fertige Routen verwenden.

Planen Sie Ihre Reise mit der Navigationsoption

Sie können eine neue Route mit der Navigationsoption der App erstellen. Dieses Tool soll Ihnen dabei helfen, eine optimale Route gemäß Ihrem Fahrzeug, Ihren persönlichen Bedürfnissen usw. zu finden. Sie können ein Auto, ein Fahrrad, einen Fußgängermodus usw. wählen und bestimmte Straßentypen vermeiden usw. Sobald die Route gebaut ist, erhalten Sie eine Turn-by-Turn Sprachführung. Wenn Sie den Track Ihrer Bewegung speichern möchten, aktivieren Sie einfach die Streckenaufzeichnungs-Erweiterung und starten Sie einen neuen GPX-Track, bevor Sie fortfahren.

Planen Sie Ihre Reise mit einem GPX-Track

Sie können einem zuvor erstellten GPX-Track folgen. Egal, ob es sich um Ihren eigenen Track handelt oder Sie ihn heruntergeladen haben, es ist einfach zu tun. Wechseln Sie einfach zum Navigationsmenü, indem Sie auf die Schaltfläche Navigation tippen, und wählen Sie dann Optionen - GPX-Route. Sie können einen GPX-Track auswählen, den die Navigation folgen soll. Tippen Sie auf Los und beginnen Sie Ihre Reise.

Entfernungsmesser

'Entfernungsmesser' ist ein praktisches Werkzeug, mit dem Sie Entfernungen zwischen mehreren Punkten auf der Karte messen können - sowohl durch gerade Linien als auch durch berechnete Routen.

Diese Funktion ist auch für Ihre Reiseplanung sehr praktisch: Fügen Sie die Punkte, die Sie besuchen möchten, zur Karte hinzu, berechnen Sie die Routen zwischen ihnen oder verbinden Sie sie einfach mit geraden Linien, wenn Sie nur ein allgemeines Verständnis der Reise benötigen, und exportieren Sie die Ergebnisse als GPX-Datei und folgen Sie sie später. Eine Route aus Favoriten erstellen

Android

Sie können Ihre Favoriten zu einer Route zusammenfügen. Dies ist besonders praktisch für Urlaube und Reisen zu neuen Orten: Fügen Sie die Punkte, die Sie besuchen möchten, zu den Favoriten hinzu, und bauen Sie eine Route, die alle Orte umfasst. Gehen Sie dazu einfach zu Meine Orte - Meine Favoriten und tippen Sie auf das Flaggensymbol unten. Jetzt können Sie die Orte auswählen, die Sie besuchen möchten. Tippen Sie danach noch einmal auf das Flaggensymbol. Alle ausgewählten Orte werden als Kartenmarkierungen auf der Karte angezeigt. Tippen Sie auf die Schaltfläche ⋮, um sie der Route hinzuzufügen. Sie können die Markierungen auch von weitesten bis zum nächstgelegenen oder umgekehrt sortieren. Um Markierungen zu bestellen oder die Reihenfolge zu ändern, tippen Sie einfach auf die rechte obere Schaltfläche auf dem Kartenmarkierungenbildschirm. Sie können auch Wegpunkte hinzufügen, nachdem die Route berechnet wurde. Tippen Sie einfach auf die Karte und wählen Sie ‘Als letztes/erstes/weiteres Zwischenziel’ hinzufügen. Sie können danach die Wegpunkte ordnen. Wählen Sie im Navigationsmenü Ihren Ausgangspunkt. Sie können die Route von Ihrer aktuellen Position aus, einem Favoriten auf der Karte, dem ausgewählten Punkt auf der Karte oder einer bestimmten Adresse starten.

iOS

Um Ihre Favoriten als Wegpunkte hinzuzufügen, gehen Sie zu Meine Ausflüge - Neue Ausflug erstellen und tippen Sie dann manuell auf die Favoriten, um sie als Wegpunkte hinzuzufügen. Dann drücken Sie Speichern.

Ausflugsplanung für iOS

OsmAnd für iOS bietet ein Plugin zur Ausflugsplanung. Damit können Sie Ihre Wegpunkte speichern und ihren Standort anzeigen. Um eine neue Reise zu erstellen, rufen Sie das Menü Meine Ausflüge auf, wählen Sie Neue Ausflug erstellen und fügen Sie durch lange Berührungen Wegpunkte hinzu. Sie können dem Wegpunkt auch Details hinzufügen, z. B. seinen Namen, die Farbe, die diesen Wegpunkt angibt, seine Gruppe (Sie können eine Gruppe von Wegpunkten erstellen: Arbeitsorte, Lieblingsorte in der Stadt usw.) und eine Beschreibung. Dann klicken Sie auf Speichern. Sie können Ihre gespeicherte Reise im Menü Meine Ausflüge anzeigen.

Versuchen Sie das Routenverwaltungsmenü, um die Punkte zu sortieren. Tippen Sie im Navigationsmenü auf das Flaggensymbol und ziehen Sie den Wegpunkt, den Sie verschieben möchten, mit der Taste '≡' nach oben oder nach unten oder streichen Sie den Namen des Wegpunkts nach links, und tippen Sie auf die Taste ‘x’. Sie können auch Wegpunkte aktivieren oder deaktivieren, um Ihre Route in diesem Menü anzupassen, oder unendlich viele Wegpunkte hinzufügen.

Sie können auch zwischen zwei Punkten navigieren. Wählen Sie dazu einen Startpunkt und ein Ziel und beginnen Sie Ihre Reise. Passen Sie Ihre Route an, indem Sie die Steuerrad-Taste drücken und die gewünschten Optionen auswählen oder zum Menü "Navigationseinstellungen" gehen.

Die hinzugefügten Wegpunkte werden in einer Informationsleiste oben auf dem Bildschirm angezeigt. Jetzt können Sie Wegpunkte anzeigen oder löschen, indem Sie in der Informationsleiste auf die Schaltfläche '^' drücken und nach links streichen. Tippen Sie auf den Punkt, den Sie erreichen möchten, um die Entfernung zu ihm zu sehen und die Position auf der Karte anzuzeigen. Bitte beachten Sie, dass die Punkte durch gerade Linien miteinander verbunden sind und Ihnen helfen sollen, die Richtungen für Ihre Reise allgemein zu verstehen. Leider bietet dieses Tool noch keine Navigation entlang der Straßen. Sie können jedoch die einfache Navigation zwischen zwei Punkten anhand der folgenden Anweisungen ausprobieren. Um einen Wegpunkt als besucht zu markieren, streichen Sie den Namen des Wegpunkts nach links und drücken Sie die Häkchen-Schaltfläche ('✓').
