Favoriten benutzen

In OsmAnd können Sie den Favoriten Orte hinzufügen, um deren Standort zu speichern und die Navigation noch einfacher zu gestalten. Um einen Ort zu den Favoriten hinzuzufügen, tippen Sie einfach auf die Karte, und wählen Sie dann “Zu Favoriten hinzufügen.” Danach können Sie es in Sekunden in Ihrer Favoritenliste finden. Sie können Gruppen von Favoriten erstellen, indem Sie ihnen beim Hinzufügen einen bestimmten Namen im Untermenü "Kategorie" geben. Das Hinzufügen eines Namens und einer Beschreibung zu einem neuen Favoriten Ort ist ebenfalls möglich.

Favoriten auf der Karte anzeigen

Um alle Ihre Favoriten auf der Karte anzuzeigen, rufen Sie einfach die Übersichtsseite auf und tippen Sie im Menü Favoriten auf “Alle anzeigen”. Tippen Sie anschließend auf die Optionsschaltfläche und wählen Sie “Auf Karte anzeigen”. Sie können auch die Farbe der Gruppe auswählen. Dies funktioniert gut mit verschiedenen Favoritengruppen.

Eine andere Option, Ihre Favoriten anzuzeigen, befindet sich im Menü Karte konfigurieren - Anzeigen - Favoriten. Jetzt sehen Sie alle Ihre Favoriten auf der Karte.

Sie können ganz einfach die Favoriten als Wegpunkte für eine Reise auswählen: Tippen Sie einfach auf die Navigation-Schaltfläche und fügen Sie die benötigten Punkte der Reise hinzu. Tippen Sie dann auf “Los”.

Um einen Favoriten oder eine Gruppe von Favoriten zu löschen, drücken Sie einfach die Mülleimer-Taste in Meine Orte - Favoriten, markieren Sie diejenigen Favoriten, die Sie löschen möchten, und tippen auf die Taste "Löschen".
