OsmAnd-Funktionen

Die OsmAnd-App ist mit einer Vielzahl von Standardfunktionen ausgestattet, es besteht jedoch die Möglichkeit, noch mehr hinzuzufügen. Es stehen häufig kostenlose Plugins zur Verfügung, die in bestimmten Situationen helfen sollen. Sie können ein Tool für bessere Parkmöglichkeiten, Wintersportorte, Karten usw. herunterladen. Hier werden einige davon kurz besprochen.

    • Online-Karten (Erweiterung)

    Mit dem Online-Karten-Erweiterung können Sie neue Informationen Schichten in Ihre Karte einfügen. Wählen Sie die Kartenquelle selbst aus und fügen Sie der Hauptansicht der Karte bis zu zwei Ebenen hinzu. Das Plugin ist für kleinere Gebiete hilfreich, wenn Sie keine vollständige Landkarte des Landes benötigen.

    Preis: Kostenlos Einzelheiten

    Höhenlinien und Relief-Schattierungen (Erweiterung)

    Mit dieser Erweiterung können Sie Informationen zu den Höhen des Geländes (Höhenlinien) anzeigen und deren Visualisierung durch Aktivieren der Relief-Schattierungen anzeigen. Wenn Sie diese beiden Funktionen kombinieren, erhalten Sie sowohl die Informationen in Zahlen als auch die visuelle Darstellung. Regelmäßige, Kontur- und Relief-Schattierungen-Karten sind drei verschiedene Arten von Downloads.

    Kostenpflichtig Einzelheiten Herunterladen

    Streckenaufzeichnung (Erweiterung)

    Mit dieser Funktion können Sie Ihre Reise aufzeichnen und als GPX-Datei speichern. Es verwendet das GPS Ihres Handys, um Ihre Bewegung zu verfolgen. Sie können die zurückgelegte Route anzeigen, an das OpenStreetMap-Projekt senden oder in externe Dienste hochladen.

    Preis: Kostenlos Details

    Ski-Karten (Erweiterung)

    Diese Karten sollen Ihnen bei der Navigation an Wintersportorten helfen. Sie enthalten Informationen über Sportanlagen, Skipisten und sogar die ungefähre Bewertung der Komplexität der Strecke.

    Preis: Kostenlos Details Download

    Seekarte (Erweiterung)

    Genau wie übliche OpenStreetMap-Karten sind Seekarten Open-Source-Karten. Sie geben Ihnen Informationen über Meere, Flüsse, Häfen usw., einschließlich der Segelrouten, gefährlicher Gebiete, Bereiche, in denen das Segeln oder Docken erlaubt ist oder nicht erlaubt ist usw. Sie sind kein Ersatz für offizielle Seekarten, die Segler auf Schiffen haben müssen. Sie können aber ein großartiges zusätzliches Werkzeug sein.

    Preis: Kostenlos Details Download

    Audio- / Video-Notizen (Erweiterung)

    Mit diesem Plugin können Sie Erinnerungen während der Fahrt speichern und mit Ihren Routen verbinden. Nehmen Sie Fotos, Videos oder Audionotizen auf. Sie können sie auch exportieren, um sie in einer anderen Software anzuzeigen.

    Preis: Kostenlos Details

    OSM-Bearbeitung (Erweiterung)

    Möchten Sie die Karte selbst bearbeiten? Werfen Sie einen Blick auf die OSM-Bearbeitungs-Erweiterung. OsmAnd verwendet die Karten, die im Projekt OpenStreetMap erstellt wurden, eine kollaborative Gemeinschaft von Freiwilligen, die Karten erstellen und aktualisieren. Wenn Sie es selbst ausprobieren möchten, aktivieren Sie einfach die Erweiterung und aktualisieren Sie die Informationen über Points of Interest an einem beliebigen Ort. Um weitere Möglichkeiten zu erhalten, nehmen Sie an OpenStreetMap teil.

    Preis: Kostenlos Details

    Entfernungsmesser (Erweiterung)

    Mit diesem Tool können Sie die Entfernung zwischen zwei oder mehr Punkten messen, die durch gerade Linien verbunden sind, und GPX-Dateien erstellen und bearbeiten.

    Preis: Kostenlos Details

    Parkposition (Erweiterung)

    Es ist ein Plugin, das Sie an Ihren genauen Parkplatz und die Zeit erinnert, wenn Ihre Parkzeit abgelaufen ist (fakultativ). Dies ist besonders hilfreich, wenn Sie an einen anderen Ort reisen.

    Preis: Kostenlos Details Download

    OsmAnd-Entwicklung

    Dieses Plugin ermöglicht das Entwickeln und Debuggen. Sie können Sprachanweisungen testen, Standorte imitieren und die Renderleistung anzeigen. Dieses Plugin ist für Softwareentwickler. Gehen Sie zu Erweiterungen - OsmAnd-Entwicklung und wählen Sie Aktivieren.
