Routensimulation

Die OsmAnd-Entwicklungs-Erweiterung benötigen nur Softwareentwickler. Sie können Navigationsrouten simulieren, die Bildschirm-Render-Leistung überprüfen usw.

Wenn Sie die Routenberechnung, die Sprachsteuerung und andere Funktionen überprüfen möchten, öffnen Sie die Einstellungen - Standort simulieren und tippen Sie auf “Simulation starten”. Sie können entweder eine neue Route erstellen, indem Sie “Simulation mit berechneter Route” auswählen, oder einen GPX-Track dafür verwenden.

Um eine Navigationsroute zu simulieren, müssen Sie zuerst eine Route berechnen und nur dann diese Option verwenden.

Wenn Sie den GPX-Track auswählen, können Sie einen Track auswählen, dem Sie folgen möchten.

Einstellungen

Sie können einstellen, welche Funktionen des Plugins Sie benötigen. Gehen Sie dazu auf Erweiterungen - OsmAnd-Entwicklung - Einstellungen. Dort können Sie die Anzeige von Rendering-Performance aktivieren, die Sie bei jedem Rendern der Karte mit verschiedenen Informationen anzeigen lassen. Außerdem stehen Ihnen mehrere Navigationsoptionen und Sprachansagen Teste zur Verfügung.
