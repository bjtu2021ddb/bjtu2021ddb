Bedienungshilfen Erweiterung

Die Erweiterung für Bedienungshilfen wird entwickelt, um Sehbehinderte bei der Verwendung von OsmAnd zu helfen.

Um es verwenden zu können, brauchen Sie die Option für die System-Eingabehilfen in den Einstellungen Ihres Telefons zu aktivieren. Dann öffnen Sie die OsmAnd-App. Um die Erweiterung zu aktivieren, brauchen Sie das Erweiterungen-Menü zu öffnen: Erweiterungen - Bedienungshilfen - Aktivieren. Bitte beachten Sie, dass Sie in Erweiterungen - Bedienungshilfen - Einstellungen auswählen können, wann Sie die Erweiterung verwenden möchten. Dort können Sie auch andere Optionen aktivieren (z.B. die Benutzung eines Trackballs zur Zoomsteuerung usw.). Dort können Sie auch die Sprachgeschwindigkeit konfigurieren. Wenn Sie auf die Elemente der Benutzeroberfläche tippen, hören Sie deren Namen. Um die Karte zu verschieben, verwenden Sie zwei Finger (dies hängt von den Eingabehilfen Einstellungen des Telefons ab).

Seit Version 2.3 hat die OsmAnd-App eine Funktion womit können Sie auf einen Ort auf der Karte tippen und die Adresse dieses Ortes abrufen. Um dies im Barrierefreiheit-Modus zu tun, müssen Sie zum Hauptmenü gehen und “Standort verwenden” antippen. Tippen Sie anschließend auf das Kontextmenü, um die Adresse anzuhören. Wenn Sie auf die Karte tippen, erhalten Sie auch Informationen über die Entfernung zum nächsten Wegpunkt oder zum Zielpunkt.
