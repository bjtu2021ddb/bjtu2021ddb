Mit der OSM-Bearbeitung-Erweiterung können Sie Beiträge zu OpenStreetMap, einer globalen Community, zur Erstellung einer umfassenden Weltkarte und zur Bereitstellung aktueller Open-Source-Daten für jeden Benutzer leisten.

Starten Sie einfach die App OsmAnd Maps & Navigation und aktualisieren Sie die Informationen zu den verschiedenen Standorten.

Wie man es benutzt

Mit dem Plugin können Sie neue Objekte, sogenannte Points of Interest oder POI, auf der Karte erstellen. Wird ein neuer Shop neben Ihnen eröffnet? Fügen Sie ihn der Karte hinzu! Dein Lieblingsdenkmal fehlt auf der Karte? Platzieren Sie es auf der Karte in Sekunden. Wenn Sie Inhaber eines neu eröffneten Unternehmens sind, können Sie es auch auf der Karte anzeigen, damit die anderen Sie einfach finden können.

So wie es geht? Gehen Sie zu Erweiterungen> OSM-Bearbeitung und drücken Sie Aktivieren. Tippen Sie auf die Karte, auf der der neue POI platziert werden soll. Klicken Sie auf "Aktionen", wählen Sie "POI erstellen", geben Sie den Namen und andere Details wie Arbeitszeit, Website usw. ein. Sie müssen sich auch unter openstreetmap.com registrieren und dann Ihre OSM-Anmeldeinformationen eingeben, um Änderungen vorzunehmen.

Tracks hinzufügen

Sie können die von Ihnen erstellten GPX-Dateien zu OpenStreetMap hinzufügen. Aktivieren Sie dazu die Streckenaufzeichnungs-Erweiterung , erstellen Sie eine Route und gehen Sie zu Meine Orte> Tracks> drücken Sie die Taste '↥'. Das OSM-Bearbeitungs-Erweiterung muss aktiviert sein. Ihr Track wird https://www.openstreetmap.org/traces hinzugefügt. Sie können ihn anzeigen und andere Benutzer können ihn auch hilfreich finden. Sie können auf solche Weise auch an der Arbeit auf der Weltkarte teilnehmen. Teile einfach die Tracks, die Sie gemacht haben. Sie werden Teil des OpenStreetMap-Projekts. Stellen Sie aber sicher, dass Sie Objekte und Straßen markieren.

Ist ein Fehler auf der Karte aufgetreten?

Sie können auch über die Fehler auf der Karte berichten. Sie müssen nur auf den Ort tippen, 'OSM-Notiz erstellen' im Kontextmenü ‘Aktionen’ auswählen und dann die Informationen dazu hinzufügen. Um die OSM-Bearbeitungsebene anzuzeigen, gehen Sie zu ‘Karte konfigurieren’ und aktivieren Sie OSM-Notizen (online). Sie sehen alle Notizen in dem jeweiligen Gebiet.
