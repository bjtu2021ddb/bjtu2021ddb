Составить маршрут

Функция 'Составить маршрут' позволяет измерять расстояния на карте, создавать треки GPX или добавлять новые сегменты к своим собственным, привязать трек к ближайшей разрешенной дороге с помощью одного из ваших профилей навигации. Этот интерфейс заменил старый инструмент «Измерение расстояния». Мы добавили новые функции и создали совершенно новый вид для максимального удобства создания маршрута для вашей поездки.

Чтобы использовать эту функцию, просто перейдите в главное меню OsmAnd и нажмите «Составить маршрут». Вы увидите карту с панелью инструментов ниже. Нажмите создать новый маршрут. Начните с нанесения первой точки на карту и добавления следующих (одну или несколько). Для этого нажмите кнопку «Добавить» при перемещении центра экрана в нужную точку или просто коснитесь места на карте. Вы увидите общее расстояние на панели инструментов. Вы также можете отменить или повторить шаги с помощью кнопок со стрелками.

'Составить маршрут' режим позволяет использовать разные типы навигации для каждого сегмента и привязать любой трек к дорогам. Что это означает?

Например. вы хотите проложить маршрут для своего путешествия по эндуро. Первая часть вашего путешествия - это дорога от вашего дома к бездорожью, вторая часть - грунтовая дорога, а третья часть - дорога домой.

    Перейдите в Главное меню -> Составить маршрут. Выберите "Создать новый маршрут", затем щелкните значок профиля и выберите профиль дороги для первой части нашей поездки. После этого добавляем начальную и конечную точки. Первый сегмент построен.

    Во второй части нашей поездки мы нажимаем на значок профиля и выбираем наш профиль навигации Enduro (предпочитать грунтовые дороги) и «Следующий сегмент» (только следующий сегмент будет пересчитан с использованием выбранного профиля). После этого мы выбираем точки для нашей поездки по бездорожью.

    Для последней части (возвращение домой) выбираем профиль Road и добавляем конечную точку нашей поездки. Нажмите кнопку «Готово» и сохраните (и переименуйте) наш файл GPX с поездкой.

    Теперь щелкаем по нашему треку, меняем его Внешний вид. Нажмите кнопку «Маршрут» и начните путешествие, используя трек GPX, который мы построили в инструменте «Составить маршрут».

    Затем вы можете открыть существующий трек, импортировать трек или открыть последний отредактированный трек.

    Когда вы жмете "Открыть существующий трек" вы можете выбрать файл трека для открытия.

    Теперь выбранный вами трек - в режиме планирования маршрута. Где вы можете изменить его для поездки. Вы можете нажать кнопку «Опции» и выбрать для нее любые параметры.

    В общем, вы можете построить маршрут между точками. Нажмите эту кнопку на экране или в меню параметров.

    Затем привяжите свой трек к ближайшей разрешенной дороге с помощью одного из ваших навигационных профилей, чтобы использовать эту опцию. Здесь мы выбираем профиль навигации Enduro и пороговое расстояние для расчета нашего маршрута.

    Когда маршрут построен, мы нажимаем кнопку «Применить» и «Готово», чтобы сохранить новый трек, или «Опции» -> «Маршруты» для начала нашего путешествия.

    Как видите, мы можем модифицировать любой трек для поездки. Конечно, вы можете удалять или добавлять точки в своем «Составить маршрут», если хотите изменить свой план на этом маршруте.

    Вы также можете измерить расстояние, используя Радиус рулетку. Чтобы выйти из режима измерения, нажмите кнопку «X» в верхнем левом углу экрана.

    Инструмент «Составить маршрут»: Меню -> Составить маршрут. Затем щелкните значок Профиля и выберите Навигационный профиль для первой части нашей поездки. После этого добавляем начальную и конечную точки. Первый сегмент построен.

    Во второй части нашего путешествия мы нажимаем на значок Профиля и выбираем наш профиль навигации Enduro (предпочитать грунтовые дороги) и «Следующий сегмент» (только следующий сегмент будет пересчитан с использованием выбранного профиля). После этого мы выбираем точки для нашей поездки по бездорожью.

    Для последней части (возвращение домой) выбираем Профиль дороги и добавляем конечную точку нашей поездки. Нажмите кнопку «Готово» и сохраните (и переименуйте) наш файл GPX с поездкой.

    Теперь вы можете создать свою поездку, используя один или несколько навигационных профилей, сохранив его как GPX-файл в версии OsmAnd для iOS.
