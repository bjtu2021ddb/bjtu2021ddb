Кнопка Быстрого действия

  • Для Android
  • Для iOS

    • Кнопка Быстрого действия - это удобное меню горячих клавиш, которые предоставляют прямой доступ к любым функциям, которые Вам необходимы чаще всего: их можно добавить на "панель инструментов".Таким образом, как только Вам нужно будет добавить новое Избранное, добавить место назначения навигации, включить или отключить голосовые подсказки и т.п., эти действия теперь доступны в два касания.

    Для Android

    Кнопку быстрого действия можно добавить через меню "Настроить экран". Чтобы добавить действия в меню, нажмите «Добавить действие» прямо в строке меню. Чтобы удалить действие, перейдите на экран «Настроить экран» и коснитесь меню «Быстрое действие», затем удалите все ненужные ярлыки.

    По многочисленным просьбам пользователей, начиная с версии 3.3 для Android, мы добавили действия Показать/скрыть GPX треки, Переключить дневной/ночной режим в меню Быстрого действия.

    Для iOS

    Кнопка быстрого действия добавляется через меню "Настроить Экран".

    Чтобы добавить быстрые действия в это меню, нажмите "Добавить действие" прямо в меню Быстрого действия. Чтобы удалить действие из меню, перейдите в "Настроить Экран" и нажмите "Быстрое действие", чтобы удалить оттуда все ненужные действия.
