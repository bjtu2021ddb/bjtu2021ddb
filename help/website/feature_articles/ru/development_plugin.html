Настройки

Плагин OsmAnd Отладка и разработка необходим только для разработчиков и опытных пользователей. Позволяет моделировать навигационные маршруты, проверять производительность рендеринга экрана и т.д.

Для начала зайдите в меню Плагинов и убедитесь, что плагин Отладка и разработка включен.

Вы можете обозначить, какие функции плагина вам нужны. Чтобы сделать это, зайдите в Плагины -> Отладка и разработка и через кнопку три точки выберите Настройки.

Там у вас будет возможность включить отображение информации рендеринга, это будет показывать всплывающее окно с этой информацией при каждом рендеринге карты, несколько вариантов навигации, вы также сможете тестировать голосовые подсказки.

Это меню настроек имеет несколько разделов:

  • Использовать OpenGl для рендеринга: использовать аппаратное ускорение OpenGL для рендеринга.
  • Безопасный режим: запускает приложение в безопасном режиме (замедляет работу приложения).

    Навигация

  • Выключить сложную маршрутизацию: выключает 2-фазную маршрутизацию для автомобильной навигации.
  • Умный перерасчет маршрута: для дальних поездок пересчитывайте только начальную часть маршрута.
  • Навигация OsmAnd Live: включает навигацию с учетом обновлений OsmAnd Live.
  • Общественный транспорт OsmAnd Live: включает общественный транспорт с учетом автообновлений OsmAnd Live.
  • Моделировать ваше местоположение: смоделируйте свою позицию, используя записаный трек GPX.

    Отладка и разработка

  • Отладочная информация о рендеринге: отображает производительность рендеринга.
  • Имитировать первый запуск: устанавливает признак первого запуска приложения, сохраняет все остальные настройки без изменений.
  • Показывать баннер бесплатной версии: отображает баннер бесплатной версии даже в платной версии.
  • Тест голосовых подсказок: выберите голос и протестируйте его, проигрывая объявления.
  • Буфер Logcat: позволяет проверить и поделиться подробными логами приложения.

    Информация

  • Выделено памяти
  • Всего физической памяти
  • Информация A-GPS
  • Восход/Закат

Симуляция маршрута

Если вам нужно проверить, как строятся маршруты, как работает голосовое управление и т.д., вам необходимо перейти на панель управления, найти “Моделировать ваше местоположение” и нажать “Начать моделирование”. Вы сможете либо построить новый маршрут, выбрав “Симуляция использования рассчитанного маршрута”, либо использовать для этого трек GPX “Симуляция использования трека GPX”.

Чтобы смоделировать маршрут навигации, сначала необходимо рассчитать маршрут, а затем нажать Опции -> Моделировать навигацию в меню навигации. После этого нажмите Старта для симуляции навигации.

Если вы выберите Маршрут GPX, вам нужно будет выбрать GPX трек и можно будет следовать по нему.
