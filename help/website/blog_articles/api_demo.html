OsmAnd API Demo

May 6, 2016

OsmAnd API Demo allows you to test integration with OsmAnd and functioning of the main features.

OsmAnd API features:

  • Adding favorites and markers to the map
  • Creating audio/video/photo notes
  • Starting and stopping GPX track recording
  • Importing GPX tracks into OsmAnd and navigating along them
  • Navigation between locations

More about features and how to work with them: https://github.com/osmandapp/osmand-api-demo

To test OsmAnd API Demo options, you need to install one of OsmAnd Maps versions.
