OsmAnd 3.10 (iOS)

Декабрь 04, 2019

Мы рады сообщить, что iOS-версия OsmAnd стала еще удобнее!

Большое спасибо всем пользователям iOS за выбор OsmAnd для ваших приключений! Вы можете прочитать далее, что нового в Osmand 3.10 или обновите приложение здесь и протестируйте все изменения самостоятельно.

• Новый переработанный экран подготовки навигации

Мы улучшили экран подготовки навигации. Теперь вы можете добавить «Дом», «Работа», пункт назначения. Короткий путь к меню настроек.

Когда ваша поездка будет готова, вы можете увидеть график высоты / уклона, который вы можете масштабировать для всей вашей поездки.

• Добавлена поддержка онлайн-карт SQL

Мы добавили больше карт в «Онлайн карты» (Общее меню-> Карта-> Тип карты-> Онлайн-карты). Эти карты также доступны для Наложения/Подложки.

Далее вы можете добавить свои онлайн-карты SQL для Наложения/Подложки. Нажмите, чтобы выбрать файл в формате SQL и «Копировать в OsmAnd Maps». Теперь вы можете выбрать его в меню Наложения/Подложки.

Вы можете включить регулировку видимости на экране для редактирования прозрачности ваших Наложения/Подложки на карте.

• Объединены все параметры для управления контурными линиями на карте на одном новом экране

Вы можете управлять настройками контурных линий на новом экране. Перейдите в Общее меню-> Карта-> Стиль карты-> Контурные линии.

Мы добавили больше цветовых схем для контурных линий.

• Добавлены еще два Быстрых Действия для быстрого включения / выключения контурных линий

В этом релизе мы добавили новые варианты Быстрого Действия: Показать / Скрыть контурные линии.

• Обновлена базовая карта

Теперь карта обзора мира имеет более подробную дорожную сеть и стала меньше по размеру.

• Исправлена проблема с исчезновением номеров линейки

Как вы знаете, в OsmAnd есть инструмент под названием «Линейка радиуса» (Общее меню-> Настройка экрана-> Правая панель-> Линейка радиуса). Была ошибка с количеством кругов для инструмента Линейка радиуса. Теперь вы можете видеть все круги без этой ошибки.

• Полностью удалена аналитика Firebase

Так как мы следуем Политике конфиденциальности - в этом релизе мы удалили аналитику Firebase. Теперь ни одно из наших приложений не собирает личную информацию наших пользователей.

• Исправлена ошибка запуска OsmAnd на iOS 13

• Исправлено отображение Mapillary

• Исправлено несколько ошибок интерфейса
