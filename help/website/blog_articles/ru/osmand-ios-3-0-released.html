OsmAnd 3.0 (iOS)

Сентябрь 03, 2019

Как мы сообщали ранее, идет активная работа по улучшению iOS версии приложения. Доказательством является этот релиз. Рады представить новую версию OsmAnd для iOS.

• Добавлена функция Быстрого Действия: мгновенный доступ к различным действиям на карте с главного экрана (например, изменение стиля карты, отображение специфических POI и т.п.)

Кнопка быстрого действия это настраиваемое меню со ссылками на любые необходимые вам функции. Вы сами выбираете, какие функции добавить в это меню. К примеру, если вы хотите быстро добавить новое "Избранное" или Включить/Выключить голосовую навигацию, добавьте эти функции в меню быстрого действия, чтобы использовать напрямую с главного экрана.

Кнопка быстрого действия может быть добавлена через меню "Настройки Экрана". Чтобы добавить действия в меню, нажмите «Добавить действие» в самом меню. Чтобы удалить действие, перейдите на экран «Настройка» и коснитесь меню «Быстрые действия», затем удалите все ненужные ярлыки.

• Улучшение контекстного меню: добавлена возможность скачивать отсутствующие карты регионов прямо из контекстного меню

Теперь нажимая по точке на карте, если у вас нет подробной карты этого региона, вы можете нажать по кнопке в меню, чтобы быстро загрузить её.

• OpenStreetMap редактирование: на экране продвинутого редактирования добавлены подсказки для тегов и значений

Добавлены подсказки тегов и значений на продвинутом экране редактирования для вашего удобства. Теперь можно быстро выбирать теги из существующих. Дополнительная информация.

• Улучшено UI записи маршрута

Исправлено UI записи трека. Теперь по кнопке записи видно, когда запись включена, а когда отключена.

• Исправлена проблема с навигационными оповещениями, которые не обновлялись

Голосовые и визуальные оповещения теперь работают корректно во время навигации.

• Другие улучшение и исправления нераспространенных ошибок.
